# Syntaxe
Quelques cours de syntaxe formelle.

Dans les dérivations TAG proposées dans ce repo, nous utilisons les notations françaises plutôt que les notations internationales (par exemple "GN" plutôt que "NP"), hormis pour "S" (*Sentence*).

Nous modélisons les verbes transitifs directs à deux arguments par :
```latex
\begin{tikzpicture}
\Tree [.S [.NP↓1↓1​ ] [.VP [.V verbe ] [.NP↓2↓2​ ] ] ]
\end{tikzpicture}
\end{center}
```
